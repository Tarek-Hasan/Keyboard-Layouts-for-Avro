# Keyboard Layouts for Avro
Additional keyboard layouts for Avro

**Layouts:**

- UniJoy 5

![KB-Bengali-UniJoy](https://user-images.githubusercontent.com/27731012/188329390-71789b79-3768-47b6-b0e3-f19002ecc112.svg)

UniJoy 5 is a modified version of [UniJoy](https://www.ekushey.org/?page/uni_joy_layout) developed by Ekushey Team. UniJoy is a Bijoy-compatible keyboard layout. It's modified to be more modern and easy to use. Version 5 refers to Avro keyboard layout filetype version.
